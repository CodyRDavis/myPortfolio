My Works

DiceBag

A project I made to streamline the Dungeons and Dragons/Pathfinder experiance. Create custom dice to represent your rolls, and the face of each dice updates with your total rolled. fast, effective, and takes the math out of playing this RPG to streamline combat.

Essentials

This was my first group project at bootcamp. we utilized a number of APIs and fethched data based on their search. we present the user with information about the area they search. This was intended to give a snapshot of important places around the users searched area.

Giphy

My first project using API calls. Takes a search term from the user and goes to Giphy to get images related to that search and displays them.

Get Out

My take on a choose your own adventure game. This was my first project playing around with timers to handle some function calls. The timer was used to track how much time the user had left before getting a game over. the only way to clear the timer is to escape the house before midnight.

RPG

A simple RPG which loosely uses D20 rules. The user is able to pick a character and go into the dungeon fighting a random monster untill the hero dies. there is no way to win, just luck guiding our hero to see how deep into the dungeon they can go.
